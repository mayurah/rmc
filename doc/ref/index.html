Refinitiv Management Classes 2.2.0 Documentation

Overview

The Refinitiv Management Classes (RMC 2.2.0) provides a stable implementation of managing and monitoring solutions: RMC Publisher and RMC Consumer API. The RMC API uses the concept of shared memory which offers fast and efficient benefits. The publisher applications are able to use the API without affecting the performance of the application, while the consumer applications are able to use the API to monitor the publisher application and receive notifications as the information changes.
